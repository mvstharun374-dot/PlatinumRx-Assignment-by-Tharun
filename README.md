## 🟢 SQL Tasks

### 🔹 Hotel Management System

* Created database schema (users, bookings, items, booking_commercials)
* Performed analysis:

  * Last booked room per user
  * Monthly billing calculations
  * High-value bills filtering
  * Most & least ordered items
  * Second highest billing customers

### 🔹 Clinic Management System

* Created schema (clinics, customers, clinic_sales, expenses)
* Performed analysis:

  * Revenue by sales channel
  * Top customers
  * Monthly profit/loss analysis
  * Most & least profitable clinics
